# React Blog Form

## Milestone 1
- Creare una pagina che visualizzi una lista di articoli, mostrandone solo il titolo e il contenuto.
 
## Milestone 2
- Aggiungiamo in pagina un semplice form con un campo input in
  cui inserire il titolo di un nuovo articolo del blog e il suo contenuto
  (potete provare a usare il tag textarea).
- Al submit del form, mostrare la lista degli articoli aggiornati.
